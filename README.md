# Calculadora Inteligente em Python

Uma calculadora interativa desenvolvida em Python para terminal Linux/macOS, com suporte a histórico, comandos inteligentes e tratamento de erros.

---

##Funcionalidades

- Operações matemáticas básicas
- Histórico de cálculos
- Limpar último caractere digitado
- Reiniciar calculadora
- Apagar histórico
- Encerrar programa com confirmação
- Tratamento de erros
- Interface colorida no terminal

---

## Tecnologias utilizadas

- Python 3
- Terminal Linux/macOS
- Shell Script (.sh)

---

##Estrutura do projeto

```bash
calculadora-inteligente/
│
├── calculadora.py      # Código principal da calculadora
├── calculadora2.sh     # Script para executar o programa
├── calculadora2.txt    # Anotações/comandos auxiliares
└── README.md           # Documentação do projeto
```

---

## Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/calculadora-inteligente-.git
```

### 2. Acesse a pasta do projeto

```bash
cd calculadora-inteligente-
```

### 3. Dê permissão ao script

```bash
chmod +x calculadora2.sh
```

### 4. Execute o projeto

```bash
./calculadora2.sh
```

---

## Comandos disponíveis

| Comando | Função |
|---|---|
| H | Exibe o histórico |
| D | Remove o último caractere |
| A | Apaga todo o histórico |
| R | Reinicia a calculadora |
| S | Encerra o programa |
